# LFPro Tasks

Sistema de gestao de tarefas colaborativo para a LFPro, construido com React + TypeScript + Vite + Supabase.

## Stack

- React 18 + TypeScript 5.8 + Vite 5
- shadcn-ui (Radix UI) + Tailwind CSS 3
- Supabase (PostgreSQL + Auth + Realtime)
- TanStack React Query v5

## Desenvolvimento

```bash
npm install
npm run dev        # porta 8080
npm run build      # build de producao
npm run test       # testes (vitest)
npm run lint       # eslint
```
